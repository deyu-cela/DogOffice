# Game Overview

玩家扮演狗狗公司老闆，目標是撐越多天、員工越多、資金越穩越好。

## 核心迴圈

1. 看 [[Candidate Generation|候選人履歷]] → 決定 [[錄用]] 或 [[婉拒]]
2. 每 7 秒 × [[速度倍率]]（1x/2x/3x）推進一天，觸發 [[Daily Settlement|每日結算]]
3. 用賺到的錢 [[Office Levels|升級辦公室]] 或買 [[Shop Items|道具]]
4. 員工滿 2 人可開啟 [[Mini Games/Training|培訓]]，滿 3 人可開啟 [[Mini Games/Frisbee|陪玩]]
5. 員工表現差可啟動 [[PIP System]]
6. 撐到 [[Bankruptcy]] 條件觸發就遊戲結束
